# Carpe for Windows

## System Requirements

- **Windows 10 or higher**

## Prerequisites

**Important**: Before installing Carpe, you must first install Microsoft Edge WebView2. This dependency is required for Carpe to function properly.

**[Download Microsoft Edge WebView2](https://developer.microsoft.com/en-us/microsoft-edge/webview2/consumer/)**

*Note: We're working on including this dependency automatically in future releases.*

## Download and Installation

You can download the latest Windows installer from our releases page:

**[Download Carpe for Windows](https://github.com/0LNetworkCommunity/carpe/releases/download/v1.0.7/carpe_1.0.7_x64_en-US.msi)**

### Installation Steps:
1. Download and install Microsoft Edge WebView2 (link above)
2. Download the Carpe `.msi` installer
3. Run the installer and follow the prompts
4. Launch Carpe from your Start menu or desktop shortcut

## Next Steps

After successful installation and launch, continue with our [Carpe Usage Guide](usage.md) to get started.
